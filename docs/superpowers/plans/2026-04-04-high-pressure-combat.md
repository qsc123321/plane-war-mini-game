# High-Pressure Combat Implementation Plan

Goal: upgrade the current shooter into a short-session action game focused on dodging pressure and explosive kill windows.

1. Extend game state for combo, overdrive, particles, wave labels, and victory handling.
2. Replace loose random pacing with staged pressure waves and a final-boss timer.
3. Add enemy firing patterns, graze charging, combo scoring, and Overdrive activation.
4. Improve combat feedback with particles, screen flash, shake, and stronger HUD cues.
5. Verify via build, run startup, and balance tuning.
