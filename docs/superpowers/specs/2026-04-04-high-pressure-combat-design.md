# High-Pressure Combat Design

Goal: turn the current shooter into a 2-3 minute dodge-heavy run with burst windows earned through skillful play.

- Regular enemies must create bullet pressure before the boss.
- Grazing bullets and close kills charge Overdrive.
- Overdrive is a short burst state, not a passive screen wipe.
- Combo and kill feedback must be visible and immediate.
- The run ends on victory or defeat, with the final boss arriving around the 135 second mark.
